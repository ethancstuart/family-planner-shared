# Contributing

Thanks for your interest in contributing to Family Planner!

## Getting Started

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make your changes
4. Run checks: `npx tsc --noEmit && npm run lint && npm test`
5. Commit with a descriptive message
6. Open a pull request

## Development Setup

See the [README](README.md) for setup instructions.

## Code Style

- TypeScript strict mode
- ESLint + Prettier (run `npm run format` before committing)
- Follow existing patterns in the codebase

## Reporting Issues

Open a GitHub issue with steps to reproduce, expected behavior, and actual behavior.
